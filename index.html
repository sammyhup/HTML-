Conceitos Básicos de Tecnologia da Informação

Hardware & Software

Hardware

Hardware é a parte física de um dispositivo
eletrônico. São as peças e os periféricos externos.
Podemos dizer que o Hardware é tudo aquilo que
podemos tocar em um dispositivo eletrônico.

Software

Software é a parte lógica de um dispoditivo
eletrônico. São os programas de computador.
Podemos dizer que o Software é tudo aquilo que eu
não posso tocar em um dispositivo eletrônico.

Periférico

			Periférico é um didpositivo eletrônico utilizado
			externamente a um computador. Ele é utilizado 
			pelos seres humanos para operar um computador.
			Por exemplo, o mouse, o teclado e o monitor.
